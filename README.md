# PhishGuard AI

PhishGuard AI is a machine learning-based phishing email detection web application developed as a cybersecurity project. The application analyzes email content and predicts whether an email is safe or potentially phishing.
The goal of this project is to demonstrate how machine learning can be applied to cybersecurity problems and help users identify suspicious emails before they become a security threat.

## Features

- Detects phishing and legitimate emails
- Machine learning-based classification
- Confidence score for predictions
- Email risk analysis
- Interactive dashboard
- Model performance metrics
- Confusion matrix visualization
- PDF report generation
- Responsive and user-friendly interface

## Technologies Used

- Python
- Flask
- Scikit-Learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript
- Chart.js
- FPDF2
